# Card-manufacturing-analysis
This project focuses on transforming traditional card manufacturing operations by replacing manual logbook-based recording with a fully digitized data management system. The existing process relied heavily on manual entries, which led to operational inefficiencies, data inaccuracies, and difficulty in performance analysis. To address these challenges, a digital solution was designed to automate data collection, storage, and analysis, enabling real-time monitoring of manufacturing activities. The system improves operational efficiency by minimizing human errors and simplifying data tracking while maintaining a strong emphasis on user-friendliness to ensure adoption across all organizational levels. By leveraging structured data analysis and automation, the project aims to support better decision-making, optimize production workflows, and enhance overall business performance. The expected business outcomes include a 10% increase in profit margins and a 20% growth in revenue within the first year of implementation.
